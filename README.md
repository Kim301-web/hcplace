# hc place 🏞️ - 홍천군 지역 소개 웹사이트

홍천군의 맛집과 숨은 장소를 인터랙티브 지도로 소개하는 웹사이트입니다.

## 주요 기능

### 🏠 사용자 (방문자)
- 🗺️ 인터랙티브 지도에서 지역 둘러보기
- 🍽️ 맛집 카테고리별 검색 (홍천읍, 서면, 두촌면, 화촌면, 영귀미면, 서석면)
- 🏞️ 숨은 장소 탐색 (박물관, 역사, 자연)
- 📍 각 장소 클릭 시 상세 정보 확인
- 🖼️ 장소별 사진 보기

### ⚙️ 관리자
- ➕ 새로운 장소 추가
- ✏️ 기존 장소 정보 수정
- ❌ 장소 삭제
- 🏷️ 카테고리 관리
- 📍 지도에서 클릭하여 좌표 자동 입력
- 🖼️ 사진 URL 추가

## 사용 방법

### 🗺️ 지도 보기
1. `index.html`을 브라우저에서 열기
2. 좌측 사이드바에서 카테고리 선택
3. 마커 클릭하면 장소 정보 확인

### 👨‍💼 관리자 패널
1. 우측 상단 "⚙️" 버튼 클릭 또는 `admin.html` 접속
2. 새로운 장소 추가:
   - 유형과 카테고리 선택
   - 장소 정보 입력
   - 지도에서 위치 클릭 (또는 좌표 직접 입력)
   - "장소 추가" 버튼 클릭
3. 기존 장소 관리:
   - 아래 목록에서 삭제 가능

## 기술 스택

- **프론트엔드**: HTML5, CSS3, JavaScript
- **지도**: Leaflet.js + OpenStreetMap
- **데이터 저장**: 브라우저 LocalStorage (자동 저장)

## 파일 구조

```
.
├── index.html          # 메인 지도 페이지
├── admin.html          # 관리자 패널
├── app.js              # 지도 기능
├── admin.js            # 관리자 기능
├── style.css           # 전체 스타일
└── README.md           # 이 파일
```

## 데이터 저장

- 모든 데이터는 **브라우저의 LocalStorage**에 자동 저장됨
- 페이지를 새로고침해도 데이터 유지
- 다른 탭에서도 실시간 동기화
